# catalog

SSG catalog app written using Next.js, Redux, Tailwind. [test assignment]

## Getting started

```bash
git clone git@github.com:mbredikhin/catalog.git
cd catalog
npm install
npm run dev
```

Open the http://localhost:3000/catalog/odezhda page in your browser
